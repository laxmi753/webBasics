What is HTML?

HTML is a markup language used to structure content on the web. It consists of elements represented by tags, which define the content's type and layout, such as text, images, links, tables, and more.

Why HTML?

HTML serves as the foundation of the web because:

  • It provides a universal way to structure and display content.
  • It is easy to learn and use, enabling anyone to create web pages.
  • It's supported by all browsers, ensuring compatibility across devices.
  • It interacts well with other technologies, such as CSS (for styling) and JavaScript (for interactivity).

HTML syntax:

  • <!DOCTYPE HTML>: Declares the document type.
  • <HTML>:The root element of the HTML document.
  • <head>:Contains metadata about the document (like the title).
  • <title>:Sets the title of the web page (visible in the browser tab).
  • <body>:Contains the main content displayed on the page.
  • Tags like <H1>and <P>represent headings and paragraphs, respectively.

HTML tags:

  • <h1>to<h6>:Define headings, from largest(<h1>)to smallest(<h6>).
  • <P>: Defines a paragraph.
  • <a href="URL">: Creates a hyperlink.
  • <img src="URL" alt="descrption">: Embeds an image.
  • <table>: Creates a table.
  • <tr>: Defines a table row
  • <th>: Defines a table header cell.
  • <td>: Defines a table data cell.
  • <ul>: Creates an unordered (bullet) list.
  • <ol>: Creates an ordered (numbered) list.
  • <li>: Defines a list item.

HTML attributes:

  • <id>:Assigns a unique identifier to an element.
  • <class>:Assigns one or more class names for CSS or JavaScript.
  • <style>:Adds inline CSS styles.
  • <title>:Provides additional information as a tooltip when hovered.
  • <a href="">:Defines the URL for the link.
  • <input type="">: Defines the type of input (e.g., text, checkbox, password).
