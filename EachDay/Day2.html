What is CSS?

CSS is a language used to describe the appearance of a webpage, such as layout, colors, fonts, and responsiveness. It works alongside HTML,the structure of your webpage, to enhance visual appeal.

Why CSS?

We use CSS to make websites look nice and easy to use. It helps with:

  • Design: Adds colors, fonts, and styles to make pages visually appealing.
  • Consistency: Ensures all pages of a website match in style.
  • Responsiveness: Makes websites work well on phones, tablets, and computers.
  • Efficiency: One CSS file can update the design of multiple pages at once.

How to use CSS?

  • Inline CSS: Add styles directly within the HTML element using the<style> attribute.
  • Internal CSS: Write CSS inside a <style> tag in the <head> section of your HTML file.
  • External CSS: Use an external stylesheet linked to your HTML file. This is the most efficient method.

CSS syntax:

  • Selector: Targets the HTML element you want to style (e.g h1,p , div,.class ,#id ).
  • Property: The specific style attribute you want to apply (e.g color,font-size,margin ).
  • Value: Defines how the property will be styled (e.g red,16px,10px)
